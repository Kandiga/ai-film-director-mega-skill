# Pre-Prophetic Cinema Prompt Library

Based on frame-language analysis of “הנביא החשמלי”, but written as reusable original prompt grammar. See [[reverse-prompt-engineering]] and [[cine-lock]].

## Master skeleton

```text
Experimental prophetic Hebrew art-house cinema, sacred absurdity mixed with domestic documentary realism, [specific subject], [camera grammar], [impossible collision], [light behavior], [material/texture], [composition], analog grain / VHS / archival texture only if motivated, fragmented montage logic, high contrast, volumetric haze, dream logic, post-traumatic spiritual realism, no generic fantasy clutter, no unreadable fake text.
```

## Camera grammar bank

- extreme macro human eye;
- overhead domestic POV;
- low ground angle feet/running;
- wide symbolic tableau;
- orbital Earth / cloud ocean;
- picture-in-picture hand reaching sky;
- 2x2 broadcast grid;
- CRT in foggy field;
- VHS archive insert;
- reflective window / train glass.

## Prompt examples

### Sacred absurdity

```text
Wide symbolic tableau of a sacred city landmark interrupted by one impossible animal, photoreal documentary light, centered iconic composition, quiet absurdity, high-detail architecture, no parody clutter, no readable fake text.
```

### Domestic to cosmic

```text
Overhead handheld POV of a warm yellow domestic kitchen, circular coffee surface reflecting like a planet, slight wide-angle distortion, tungsten light, mundane realism transforming into cosmic dread, no object replacement, no fantasy clutter.
```

### Media ghost

```text
Vintage CRT television standing alone in a foggy industrial field, blank screen faintly reflecting a ghostlike face, electrical pylon in background, blue-gray dusk, analog scanlines, abandoned media signal, wide static frame.
```

### Eye as portal

```text
Extreme macro close-up of a wet human eye, visible red veins and cornea reflections, shallow depth of field, reflected starburst light, unsettling prophetic witness, clinical detail mixed with mystical glow.
```
