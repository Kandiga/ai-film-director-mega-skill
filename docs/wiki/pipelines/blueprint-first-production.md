# Blueprint-First Production

A beautiful start frame is not enough. Serious AI film scenes need a blueprint layer before video spend.

## Three blueprint layers

1. Character/object blueprints — recurring people, aliens, craft, props, statues.
2. Environment/area blueprints — every major spatial zone.
3. Shot blueprints — composition, camera, motion, action, sound/SFX, forbidden drift.

This page supports [[cine-lock]] and [[etzba-emtzaeit-production-bible]].

## Versioning after correction

If the user corrects identity or intent mid-production:

1. Stop downstream video spend.
2. Mark previous assets as draft/superseded.
3. Patch the production document.
4. Regenerate governing blueprints and start frames.
5. Make a contact sheet proving corrected intent.

## QA artifacts

- locked-assets contact sheet;
- start-frame contact sheet;
- clip contact sheet;
- delivery report with files, verification, blockers, and next decision.
