# Electric Prophet Frame Analysis Raw Note

Frame-level analysis of “הנביא החשמלי” produced reusable lessons for [[pre-prophetic-cinema]] and [[reverse-prompt-engineering]].

## Reusable lessons

- Treat the film as montage grammar, not a single static style.
- Use extreme scale jumps: eye/mouth/glass/water/smoke ↔ Saturn/Earth/cloud ocean/city panorama.
- Use one impossible collision per shot: sacred place + animal/angel, domestic kitchen + cosmic orbit, office + floating cloud, media screen + ghost/authority/data.
- Camera language matters more than object list: extreme macro, overhead POV, low ground angle, wide symbolic tableau, orbital view, picture-in-picture, 2x2 broadcast grid, VHS insert.
- Light is a subject: blown window, golden backlight, low-key smoke, overexposed safe room, starburst sun, foggy snow headlights.
- Avoid generic “surreal cinematic” prompts. Anchor the prompt in exact visual mechanics: lens, composition, symbolic contradiction, light behavior, material texture.
